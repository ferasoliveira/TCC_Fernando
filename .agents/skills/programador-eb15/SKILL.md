# Skill: Programação de Sistemas Distribuídos (Firmware)

Habilidade de desenvolvimento de baixo nível para microcontroladores aplicados à robótica.

## Domínio Técnico:
- **ESP32 S3:** Programação em C++ (Arduino IDE ou PlatformIO), gestão de FreeRTOS, servidor web assíncrono e implementação de protocolo RTDE.
- **Arduino Uno:** Otimização de timers para geração de pulsos (step/dir) e leitura de sensores I2C (AS5600) sem travar o loop principal.
- **Comunicação:** Implementação de protocolos de comunicação robustos via UART, com soma de verificação (checksum) para evitar erros de dados.
- **Lógica de Controle:** Implementação de rampas de aceleração/desaceleração e interpolação de movimentos.
