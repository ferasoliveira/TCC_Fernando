# Skill: Regras de Citação e Bibliografia

Habilidade focada na gestão de fontes e integridade acadêmica usando o sistema autor-data.

## Instruções:
- **Citação Direta:** Formatar citações curtas (até 3 linhas) entre aspas e longas (mais de 3 linhas) com recuo de 4cm e fonte menor.
- **Citação Indireta:** Priorizar a paráfrase, garantindo que a ideia do autor original seja respeitada e devidamente creditada.
- **Gestão do `ref.bib`:** - Sempre verificar se a chave de citação (ex: `\cite{oliveira2026}`) existe no arquivo `ref.bib`.
  - Ao sugerir novas referências, fornecer o código BibTeX completo para inclusão.
- **Densidade:** Manter o equilíbrio de pelo menos uma citação relevante por parágrafo em seções teóricas.
